# Software Security Course 🛡️

Цей репозиторій містить виконання лабораторних робіт з курсу "Захист програмного забезпечення". 
Проекти демонструють практичне застосування криптографічних алгоритмів, створення безпечних CLI-утиліт та роботу з системними API.

## 🛠 Технології та Інструменти
* **Мови програмування:** Rust, Bash
* **Криптографія:** OpenSSL (libcrypto)
* **Алгоритми:** AES-256-GCM, PBKDF2, RSA, SHA-256, ECDSA, ECDHE

## 📂 Структура репозиторію

### [Lab 2: AES-GCM Cryptor](./cryptor-aes-gcm)
Повноцінна CLI-утиліта, написана на **Rust** для безпечного шифрування файлів.
* **Особливості:**
  * Поблочне читання/запис файлів.
  * Генерація ключа з пароля за допомогою PBKDF2 (100,000 ітерацій).
  * Використання AES-256-GCM для забезпечення конфіденційності та цілісності (автентифікації) даних.
  * Зручний інтерфейс командного рядка (реалізовано через `clap`).
